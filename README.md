# scs-repo
Contains spack definitions for SCS lab packages.

## Installation

```bash
git clone https://github.com/scs-lab/scs-repo.git
spack repo add scs-repo
```

## Dependencies

To install spack:

```bash 
git clone https://github.com/spack/spack.git
cd spack
echo ". `pwd`/share/spack/setup-env.sh" >> ~/.bashrc
source ~/.bashrc
```
