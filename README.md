# scs-repo
Contains spack definitions for GRC packages.

## Installation

```bash
git clone https://github.com/grc-lab/grc-repo.git
spack repo add grc-repo
```

## Dependencies

To install spack:

```bash 
git clone https://github.com/spack/spack.git
cd spack
echo ". `pwd`/share/spack/setup-env.sh" >> ~/.bashrc
source ~/.bashrc
```
